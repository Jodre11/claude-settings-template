Considering building a dedicated dev box with an NVIDIA GPU. Two use cases: local ML/AI inference (talking heads, local LLMs) and gaming.

**Why:** M4 Mac lacks CUDA/TensorRT support — the ML ecosystem (Ditto, LivePortrait, etc.) is NVIDIA-only for real-time workloads. Cloud GPU rental (RunPod ~$1–2/hr) is cost-effective for burst AI jobs but unsuitable for gaming due to latency.

**How to apply:** When discussing local ML inference, game performance, or hardware purchases, factor in the dual AI + gaming use case. Cloud rental is the preferred approach for heavy/infrequent AI workloads; a local GPU only justifies itself if gaming or frequent inference tips the balance. As of Apr 2026, GPU prices are severely inflated (RTX 5080 $1,289 vs $999 MSRP; RTX 5090 $3,699 vs $1,999 MSRP) — timing matters.
