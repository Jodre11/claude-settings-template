Avoid recommending Vast.ai for anything interactive or where interruption is costly. User has first-hand experience being repeatedly outbid and preempted on the good-value servers.

**Why:** The peer-to-peer marketplace model means cheaper instances get reclaimed frequently. This was annoying enough to be a dealbreaker.

**How to apply:** When suggesting GPU cloud rental, default to RunPod or Lambda Labs. Only mention Vast.ai if the user specifically asks about cheapest options, and caveat with the preemption issue they've already experienced.
